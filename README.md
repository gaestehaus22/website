* Helle, freundliche Zimmer mit komfortablen Einzelbetten.

* Große **Wohnküche** zur Selbstversorgung.

* Gemütliches Wohnzimmer mit großem Sofa und TV.

* Terrasse für entspannte Feierabende im Freien.
